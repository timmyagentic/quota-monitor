Get started in minutes

Install Quota Monitor like any other native Mac app.

  1. 1

    Download the latest DMG

    Get the current Developer ID signed and Apple notarized disk image.

  2. 2

    Drag Quota Monitor into Applications

    Move Quota Monitor to your Applications folder like any other Mac app.

  3. 3

    Open and choose tools to track

    Select Codex, Claude Code, or both, then start monitoring quota right away.
